storage_duration

  • static 静态或线程存储期和内部链接。内存中只存在一个。
  • extern 静态或线程存储期和内部链接。外部变量，可以供所有源文件使用
  • thread_local 线程存储期。
  • mutable 不影响存储期或链接。
  • register (c++17弃用) 放在寄存器中，而非内存中， 效率更高，一般是临时变量
  • auto (c++11前) 默认都是自动变量
